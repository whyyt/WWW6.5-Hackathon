# 屿光 · IsleLight

> 记录你的精神成长，掌握属于自己的文化人生

一款面向女性的精神成长记录工具，结合轻量化 Web3 实现数据主权与文化身份，基于「岛屿-灯塔-航船」的岛塔效应理念，打造温柔共鸣、无内卷的精神手账。

## 🌊 项目信息

- **参赛赛道**：赛道2 丰盈与进化（Prosperity & Evolution）
- **核心价值**：精神财富成长 · 数据主权 · 女性文化身份构建
- **产品定位**：女性向精神成长记录工具，私密、轻量、有温度
- **技术栈**：HTML/CSS/JS + Web3 钱包连接 + SBT 合约
- **部署网络**：Avalanche 测试网（可选）

---

## 🎯 痛点与背景

当前主流内容平台存在一系列女性用户痛点：
1. **内容男性中心化**：首页推荐、封面、流量长期偏向男性叙事
2. **成长轨迹碎片化**：阅读、观影、爱好记录分散，无法形成个人成长体系
3. **分享共鸣成本高**：深度感悟难以被理解，缺乏同频交流空间
4. **数据无所有权**：平台掌握用户记录，无法导出、无法迁移

IsleLight 旨在通过轻量化产品设计 + Web3 ownership 解决以上问题。

## ✨ 产品定位

IsleLight 不是社交平台，不是评分网站，而是：
- 你的私人文化手账
- 你的链上精神身份载体
- 你与同频者温柔共鸣的空间

我们记录的不只是书和电影，而是我们如何被它们塑造。
每一次标记“看过”，都是一次自我发生。

## 🏗️ 核心功能

### 1. 多品类爱好记录
- 标记状态：想看 / 在看 / 已完成
- 星级评分体系
- 高频短评（可多次编辑）
- 深度长评（记录顿悟与思考）
- 自定义标签：类型、情绪、女性主题分类

### 2. 个人成长可视化
- GitHub 风格热力图
- 内容类型分布统计
- 月度/年度精神报告
- 一键生成分享海报

### 3. 轻量化 Web3 功能
- 钱包登录（钱包即身份，无需注册）
- SBT 灵魂绑定文化身份
- 灯塔机制：选择内容上链公开分享
- 航船机制：浏览他人灯塔，寻找精神邻居
- 数据本地存储 + 可选 IPFS 备份

### 4. 岛塔效应理念设计
- **岛屿**：每个人独立、私密的精神世界
- **灯塔**：自愿公开的感悟与思考
- **航船**：温和探索、不打扰的连接方式
- 核心：女性共生、彼此照亮、互不消耗

## 🔧 技术架构
用户钱包 → 前端页面 → 本地数据存储 → 链上 SBT 身份 → 灯塔状态上链


- 前端：AI 辅助生成，简洁温柔视觉风格
- Web3：钱包连接 + 极简 SBT 合约
- 数据：本地 localStorage 为主
- 合约：仅实现身份铸造与灯塔状态，无复杂逻辑
- 部署：测试网即可满足参赛要求
